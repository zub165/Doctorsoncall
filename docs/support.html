Common topics

  • Sign in / password — Use “Forgot password” on the login screen or change password in Settings after signing in.
  • Appointments — Book from “Book appointment” or open Appointments to view upcoming visits.
  • Provider access — Doctors register, then submit a provider application for admin approval.
  • Hospitals map — Allow Location when prompted for nearby results.
  • Medical records — Available after sign-in under Medical records & AI.
  • Subscriptions / billing — Manage plans in Client hub; purchases follow Apple’s subscription terms.

Report a problem
